# 价格监控

头部区域提供一个开关Switch（是否自动触发），如果自动触发，需要用户设置间隔时间，否则需要用户手动刷新接口

1. 内置5个平台接口（京东、淘宝、天猫、拼多多、抖音）
2. 根据用户勾选和提供的产品关键字批量调用search接口
3. 每个接口取前5条数据
4. 监控列表板块，将每个平台价格最低的数据展示出来
5. 点击比价，显示每个平台的5条数据
6. 监控列表板块每一条数据都可以手动刷新，也可以点击批量刷新按钮，一次性刷新多个平台数据

## 淘宝、天猫搜索接口

/api/taobao/search-item-list/v1

### 字段处理

所有商品在data.model.itemList

- 产品名称（itemName）
- 产品图片（https://img.alicdn.com/imgextra + picUrl）
- 产品链接（https://item.taobao.com/item.htm?id=拼接上itemId）
- 产品价格（priceYuanDouble）
- 商店名称（shopName）

## 京东搜索接口

/api/jd/search-item-list/v1

### 字段处理

所有商品在data.products

- 产品名称（title）
- 产品图片（imageUrl）
- 产品链接（click）
- 价格（price）
- 商店名称（shopName）

## 抖音电商接口

/api/douyin-ec/search-item-list/v1

### 字段处理

商品详细信息data.summary_promotions.base_model.product_info
商店详细信息data.summary_promotions.base_model.shop_info

- 产品名称（name）
- 产品图片（main_img.url_list[0]）
- 产品链接（detail_url）
- 价格（marketing_info.price_desc.price.integer 和 decimal 进行拼接，你仔细看下marketing_info.price_desc.price的字段）
- 商店名称（shop_name）
