# 部门业务工具 — 页面需求文档

**日期**: 2026-07-03
**版本**: v1.0

---

## 1. 首页

**路由**: `/`
**定位**: 平台入口，展示各工具简介，引导用户选择。

- 展示平台名称与简介
- 列出所有工具卡片（竞品分析、价格检测）
- 每张卡片包含：图标、名称、功能简述、关键词标签
- 点击卡片跳转到对应工具页面

---

## 2. 竞品分析

**路由**: `/bidding`
**定位**: 粘贴竞品电商链接，自动解析产品详情并生成多维度竞品分析报告。

### 2.1 核心流程

1. 用户粘贴电商商品链接（支持京东、天猫、拼多多）
2. 点击「开始分析」或按 Enter 触发解析
3. 系统展示该商品的竞品分析报告卡片
4. 多次分析结果累积展示，支持搜索过滤和手动删除

### 2.2 分析维度

| 维度 | 分析内容 |
|------|----------|
| 设计 | 包装评价、颜色/尺寸选项、用户外观喜恶、差异化机会 |
| 定价 | 竞品 vs 我方价格对比、分期/试用/套餐方案、定价差距 |
| 功能 | 解决场景、功能缺口、用户痛点 |
| 质量 | 易用性、耐用性、质量问题、正负面反馈 |
| 客服 | 响应风格/时间、服务亮点、常见投诉 |

### 2.3 评分体系

- 每维度 0-10 分，综合评分 = 5 维度平均值
- ≥8 分绿色、6-7 分黄色、<6 分红色
- 整体优势与机会点摘要

### 2.4 操作

- **搜索**: 按商品名称/平台/店铺筛选已有记录，点击按钮触发搜索
- **删除**: 红色垃圾桶图标 + 二次确认弹窗

## 3. 价格检测

**路由**: `/price/monitor`
**定位**: 监控竞品价格变动、跨平台比价、历史走势追踪。

### 3.1 价格监控

- 统计卡片：监控总数、降价商品数、涨价商品数
- 监控列表表格：商品名称、SKU、平台、当前价、目标价、差价、状态
- 添加监控：弹窗表单（商品名称、SKU、平台、目标价格）
- 删除监控：红色图标按钮 + 二次确认

### 3.2 平台比价

**路由**: `/price/compare`

- 商品选择器（下拉）
- 跨平台价格柱状对比图（京东/天猫/拼多多/苏宁/抖音）
- 最低价高亮标注
- 价格明细表

### 3.3 价格走势

**路由**: `/price/trend`

- 商品选择器 + 时间范围切换（7天/30天/90天）
- 历史价格折线图
- 最高价/最低价/当前价参考线标注

---

## 4. 全局约束

- 所有数据使用 Mock 数据
- 无需用户登录注册
- 左侧边栏 + 右侧内容区布局
- 支持亮色/暗色主题切换
- 支持侧边栏折叠
